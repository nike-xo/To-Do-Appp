🌙 Nikki Tech/Modern – Task Manager


A simple and modern To-Do Task Manager built using HTML, CSS, and JavaScript.

This app helps users organize daily tasks with a clean interface and smooth animations.


✨ Features




➕ Add new tasks easily


❌ Delete tasks anytime


🎨 Bright modern gradient design


📱 Responsive layout (works on phone and desktop)


⚡ Smooth fade-in animation for tasks




🛠️ Technologies Used




HTML5 – Structure of the app


CSS3 – Styling and gradient UI


JavaScript – Task functionality and interactions




📂 Project Structure


project-folder
│
├── index.html
├── style.css
├── script.js
└── README.md



🚀 How to Run the Project




Download or clone the repository


Open the project folder


Open index.html in your browser




That's it! The app will start running.
